LT8500 Description:
48 independent channel PWM generator
12-bit resolution
50MHz cascadable serial interface

3-5.5V input voltage
6-bit PWM correction
Up to 6.1kHz PWM frequency with PWMCK at 25MHz
Phase shift option to reduce input currents
